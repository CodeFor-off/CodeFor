Бесплатный курс CSS: онлайн обучение с нуля

Cascading Style Sheets (CSS) — каскадные таблицы стилей, которые позволяют оформлять содержимое страницы в соответствии с описанными правилами. Стили текста, расположение блоков на странице, анимация — всё это описывается с помощью каскадных таблиц стилей

CSS, или каскадные таблицы стилей, используются для форматирования содержимого веб-страницы на основе заданных правил. Они определяют стили текста, размещение элементов и создание анимаций.

Охватить все возможности CSS в рамках одного курса невозможно. В настоящее время CSS представляет собой мощный инструмент для задания стилей текста, блоков, анимаций. В нем используются переменные и возможности вычисления значений. Кроме того, с помощью CSS можно создать адаптивную вёрстку, которая подстраивается под различные типы устройств, включая компьютеры и мобильные устройства.

Основы разметки HTML

1. HTML разметка 2. Теги 3. Текст в HTML 4. Вложенность тегов 5. Атрибуты HTML

Для стандартизации отображения текста в браузере был разработан язык разметки HTML, который определяет правила форматирования текстовых данных. Несмотря на продолжительное время существования языка, его базовые концепции остаются неизменными даже спустя более 20 лет. В первом модуле мы изучим основные принципы HTML: теги, атрибуты, вложенность тегов, а также научимся использовать основные возможности для разметки текста с помощью параграфов.

Текст в HTML

6. Параграфы 7. Начертание текста 8. Заголовки 9. Списки 10. Вложенные списки 11. Таблицы 12. Объединение ячеек внутри строк и столбцов 13. Специальные символы HTML 14. Ссылки 15. Форматированный текст 16. Вставка компьютерного кода в HTML

HTML предлагает множество правил для работы с текстом: начиная с создания обычных блоков текста и заканчивая сложными таблицами и вложенными списками. Корректная разметка текста является одним из ключевых навыков при работе с веб-сайтом.

Медиаэлементы

17. Изображения 18. Аудио 19. Видео

HTML-разметка дает возможность вставлять изображения на страницы, а в последнем стандарте HTML5 появилась возможность работы с аудио- и видеоконтентом.

Структура HTML документа

20. Базовая структура 21. Метатеги 22. link 23. Viewportr 24. Блочные и строчные элементы

Каждая HTML-страница состоит из нескольких основных блоков, которые содержат различные наборы данных: метатеги, заголовок страницы и контент. В стандарте HTML для каждой из этих частей предусмотрены соответствующие блоки и конструкции.

Формы

25. Формы в HTML 26. Поле для ввода текста 27. Чекбокс 28. Радиокнопка 29. Textarea 30. Список 31. Отправка формы

Формы являются ключевым компонентом HTML, поскольку обеспечивают взаимодействие пользователей с веб-сайтом. От поиска до обратных звонков и анкет — все эти функции реализуются с помощью форм.

Семантические элементы HTML5

32. Семантические элементы 33. Шапка 34. Меню 35. Уникальный контент страницы 36. Секция 37. Самостоятельная секция 38. Боковая панель

HTML5 ввело множество новых тегов для семантической разметки, целью которых является улучшение корректности обработки документа браузерами и использование режима чтения. Помимо этого, поисковые системы будут признательны, если вы поможете им лучше понимать структуру вашего сайта. В данном модуле мы будем размечать типичную веб-страницу, используя новые теги HTML5.
